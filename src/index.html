Chord:

Play acoustic guitar using your keyboard. Use left hand to change chords and right hand to either strike the strings or pluck them one by one. Chords are constructed by selecting a base tone and modifiers.

  • Base: Major
  • Base + 1: Minor
  • Base + 2: Seven

The chord keyboard resembles piano's keyboard:

  • Q: C
  • 1: C#
  • W: D
  • 2: D#
  • E: E
  • R: F
  • 5: F#
  • T: G
  • 6: G#
  • Y: A
  • 7: A#
  • U: H

So for example all different C based chords:

  • Q: C
  • Q, W: Cm
  • Q, E: C7
  • Q, W, E: Cm7

To stroke, use up / down arrows. To pluck strings separately, use

  • B: String 6 (E)
  • N: String 5 (A)
  • M: String 4 (D)
  • ,: String 3 (G)
  • .: String 2 (B)
  • -: String 1 (E)
